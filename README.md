# Personal Site (GitHub Pages)

This is a minimal, clean academic homepage using GitHub Pages + **jekyll-theme-minimal**.

## Quick start
1. Create a repository named `xy-su.github.io` (public).
2. Upload these files to the repository root (or push via git).
3. In **Settings â†’ Pages**, ensure the source is set to `Deploy from a branch` on the `main` branch (root).
4. Edit `_config.yml`, `index.md`, and the pages in this repo to personalise content.
5. (Optional) Add `assets/profile.jpg` and `assets/cv.pdf` for your photo and CV.

## Customisation
- Site title/description/socials in `_config.yml`.
- Add pages (`.md`) and link them from `index.md`.
- Add Google Analytics or custom SEO via `jekyll-seo-tag` (already enabled).
